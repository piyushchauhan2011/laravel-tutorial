# Checkpoint Strategy

Use Git tags to checkpoint lesson progress:

- `lesson-00-start`, `lesson-00-done`
- `lesson-01-start`, `lesson-01-done`
- `lesson-02-start`, `lesson-02-done`
- `lesson-03-start`, `lesson-03-done`
- `lesson-04-start`, `lesson-04-done`
- `lesson-05-start`, `lesson-05-done`
- `lesson-06-start`, `lesson-06-done`
- `lesson-07-start`, `lesson-07-done`
- `lesson-08-start`, `lesson-08-done`
- `lesson-09-start`, `lesson-09-done`
- `lesson-10-start`, `lesson-10-done`
- `lesson-11-start`, `lesson-11-done`

Example:

```bash
git tag lesson-03-start
git tag lesson-03-done
```
