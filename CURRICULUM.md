# 6-8 Week Intensive Curriculum

## Week 1

- Lesson 00: Laravel CLI Foundations
- Lesson 01: CRUD and Data Modeling

## Week 2

- Lesson 02: Authn/Authz Policies and Roles
- Lesson 03: API Routes, Sanctum, Versioning

## Week 3

- Lesson 04: Background Jobs, Queues, Workers
- Lesson 05: Scheduler and Custom Artisan Tasks

## Week 4

- Lesson 06: Postgres CTEs and Analytics

## Week 5

- Lesson 07: Vue + Inertia Advanced UI
- Lesson 08: Filament Admin Panel

## Week 6

- Lesson 09: Testing (Unit/Integration/E2E)

## Week 7

- Lesson 10: Deployment with FrankenPHP on VPS

## Week 8

- Lesson 11: Capstone Production-style App

## Completion Gate

You are proficient when you can:

- Build secured CRUD + API + admin features.
- Design async workflows with jobs/queues/scheduler.
- Write and tune Postgres analytics queries with CTEs.
- Ship with automated tests and repeatable deployment.
